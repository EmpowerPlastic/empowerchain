# CosmWasm workspace

This workspace contains contracts and libraries meant to be used on EmpowerChain.

## Pre-requisites
You need to install and set up the normal CosmWasm toolchain (including cosmwasm-check).

## Build

To build:
```bash
$ cargo build
```

To build wasm:
```bash
$ cargo wasm
```

## Test

To run the tests:
```bash
$ cargo test
```

## Generate schema

To generate schema:
```bash
$ cargo schema
```
